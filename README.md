## Ekşitter (beta)
Ekşi sözlük arayüzünü Twitter'a benzetmeye çalıştım. 
Giriş yapan kullanıcılar için uygun değil, giriş yaptıktan sonra çalışan kısımlar (entry yazma vs.) henüz yazılmadı.
